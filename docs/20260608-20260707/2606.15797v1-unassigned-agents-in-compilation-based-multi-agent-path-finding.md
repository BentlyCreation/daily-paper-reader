## 摘要
基于编译的技术是多智能体路径规划（MAPF）求解器的重要分支，这得益于其模块化以及对问题非标准变体的适应性。标准MAPF的任务是将所有智能体从其初始位置导航到给定的个体目标位置，且无任何碰撞。然而，对智能体有不同要求的变体也具有相关性。其中一种变体是带有未分配智能体的MAPF（UA-MAPF）：部分智能体与标准MAPF相同，拥有初始位置和目标；其余智能体仅有初始位置而无目标——即未分配智能体。尽管未分配智能体无需到达任何目标位置，但当需要时，它们必须被移出标准智能体的路径之外，这构成了一个特殊挑战。本文表明，UA-MAPF可以通过基于布尔可满足性的问题表述，在近期的基于编译的MAPF技术中表达。具体而言，我们改编了SMT-CBS和NRF-SAT，这两种基于反例引导的抽象精化及非精化抽象的最新求解器。

## Abstract
Compilation-based techniques represent an important stream of solvers for multi-agent path finding (MAPF) due to their modularity and adaptability for non-standard variants of the problem. While in the standard MAPF the task is to navigate all agents from their initial positions to given individual goal positions without any collision, variants where a different requirement for agents is used are also relevant. Such a variant is MAPF with unassigned agents (UA-MAPF) where some agents have the same setting as in the standard MAPF with initial positions and goals while the remaining agents have the initial position but have no goal - unassigned agents. Despite unassigned agent do not need to reach any goal position they have to be moved out of the way of the standard agents if needed which represent a specific challenge. We show in this paper that UA-MAPF can be expressed in recent compilation-based techniques for MAPF based on formulating the problem as Boolean satisfiability, namely we adapt SMT-CBS and NRF-SAT, the recent solvers based on counterexample guided abstraction refinement and non-refined abstractions.

- **核心问题**：标准多智能体路径规划（MAPF）要求所有智能体从初始位置到达指定目标位置，且路径无碰撞。但在实际场景（如自动停车场、仓储机器人）中，存在一类**未分配智能体**：它们有初始位置但无目标，仅在必要时为有目标的标准智能体让路。这种变体称为UA-MAPF（Unassigned Agents MAPF）。如何高效求解UA-MAPF，尤其是将其纳入编译型求解框架，是本文的研究动机。
- **整体含义**：编译型MAPF求解器（如基于SAT/布尔可满足性）具有模块化和适应非标准变体的优势。本文通过将UA-MAPF转化为SAT问题，展示了该框架的扩展能力，从而拓展了编译型技术在更复杂任务中的应用。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将UA-MAPF问题编译为布尔可满足性（SAT）实例，利用反例引导抽象精化（CEGAR）和非精化抽象技术，适配现有的SMT-CBS和NRF-SAT求解器。
- **关键技术细节**：
  - **时间扩展图（TEG）与多值决策图（MDD）**：对标准智能体，MDD通过双向距离约束（到起点和终点的距离）裁剪不可达状态，降低公式规模。
  - **UA-MDD**：对未分配智能体，由于缺少目标，只能使用单向距离约束（仅从起点出发），因此UA-MDD比标准MDD更大，导致公式规模增加。
  - **目标函数**：除了标准SST（服务时间总和）、FUEL（总移动步数，等待不计费）、NUA（移动的未分配智能体数量），本文引入**FUEL+**（等待也计费，与标准MAPF的sum-of-costs对应）。求解器通过递增makespan和FUEL+上界来搜索最优解。
  - **约束表达**：
    - 智能体传播约束：每个智能体每一步必须出现在相邻顶点（含自身等待）。
    - 路径一致性约束：智能体不能分身或凭空出现（NRF-SAT中可省略该约束，通过后处理从DAG中提取路径）。
    - 冲突消除约束：禁止顶点冲突和边冲突。
  - **求解流程**：基于CEGAR的迭代过程，初始模型省略部分约束（如冲突消除），求解后检验是否存在冲突，若有则添加对应约束重新求解，直至无冲突或找到可行解。NRF-SAT额外保留路径一致性约束的省略（非精化抽象），从而降低求解难度。

## 6. 主要结论与发现

- **成功适配**：通过修改MDD生成（改用UA-MDD），SMT-CBS和NRF-SAT均可求解UA-MAPF，展示了编译型方法的模块化与适应性。
- **反直觉趋势**：在较大地图（random-32-32-20、room-64-64-16、ost003d）中，增加未分配智能体数量初期导致求解时间**上升**，而非下降。这是因为UA-MDD尺寸更大，抵消了问题松弛带来的好处。只有未分配智能体占比极高时，松弛效果才逐渐占优。在极小地图（empty-16-16）上，松弛效应一开始就占主导，导致时间持续下降。
- **NRF-SAT整体更高效**：由于非精化抽象减小了公式规模，NRF-SAT在各场景下通常比SMT-CBS更快，且上述趋势表现更弱。

## 7. 优点
